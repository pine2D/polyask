# 更新日志

本项目所有值得注意的变更都记录在此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)并与 `manifest.json` 一致。
发版时 release workflow 提取对应版本段作为 GitHub Release 说明，缺段落拒发。

## [未发布]

### 变更

- 升级 Gemini 快速档至 3.6 Flash

## [0.9.2] - 2026-07-21

### 修复

- 修复 Claude 深度思考档切换报错，改用 Fable 5 High

## [0.9.1] - 2026-07-21

### 变更

- Kimi 快速档由 K2.6 改为 K3 Standard（与思考档 K3 Max 同模型，仅思考强度区分）

### 修复

- 修复 Kimi 思考强度子菜单 hover 竞态导致的切档间歇性失败（每轮重取节点重发 hover）
- 修复千问已在目标模型时仍误开模型对话框导致切档极慢/不稳的问题（先读后点幂等守卫）
- 修复 Kimi 新版编辑器被合成 beforeinput 注入写死（文本上屏但无法发送、编辑器冻结）的问题：新增适配器 inject 契约，Kimi 改用 execCommand 注入
- 合成点击补齐 detail:1 拟真（与真实点击一致）

## [0.9.0] - 2026-07-20

### 变更

- 升级千问快速/思考档至 `Qwen3.8-Max-Preview`
- 升级 Claude 思考档至 `Fable 5 Max`

## [0.8.2] - 2026-07-18

### 变更

- 优化悬浮控件、弹出面板与快捷键的无障碍语义

### 修复

- 修复 Gemini 重复点击 Extended thinking 后退回普通 Pro 的问题
- 修复智谱汇总混入隐藏思考过程的问题
- 修复消息异常或空响应导致群发重复提问的问题
- 修复重页面消息悬挂导致操作队列永久卡住的问题
- 修复档位切换与发送并发时相互干扰的问题
- 修复跨窗口保存覆盖站点选择或草稿的问题
- 修复伴侣窗快速关闭时丢失最后输入的问题
- 修复归档并发操作丢项及语言切换跳选的问题
- 修复旧计时器污染新一轮群发状态的问题
- 修复 Markdown 围栏破损与危险链接输出问题
- 修复汇总问题文本串入后续编辑的问题
- 修复分组重复匹配及取消删除后选项错乱的问题

### 安全

- 严格隔离受管 popup，杜绝收编用户日常窗口

## [0.8.1] - 2026-07-17

### 修复

- 修复 Kimi 升级 K3 后无法切换深度思考档的问题
- 修复 Gemini 无法选择 Extended thinking 直达项的问题
- 修复新开群发窗口恢复旧对话的问题
- 修复 Claude Sonnet 5 快速档误操作 Thinking 菜单的问题

### 安全

- 发布包新增 SHA-256 校验文件，便于核验下载完整性

## [0.8.0] - 2026-07-13

### 新增

- 新增全套品牌图标（16/32/48/128，紫罗兰圆角方块 + 双菱形标）：128 按 CWS 规范留边，工具栏小尺寸满幅显示，与相邻扩展观感一致

### 变更

- 历史/模板/分组下拉去掉占位文案中多余的 ▾ 符号（原生箭头已有），并收窄闭合宽度消除按钮右侧空白

### 修复

- 修复控制台/编辑窗/归档窗在 Windows 任务栏与标题栏仍显示旧图标：窗口页显式声明 favicon，绕开浏览器 favicon 缓存

## [0.7.2] - 2026-07-12

### 修复

- 修复窄窗口下站点选择芯片整体消失（历史下拉无限宽挤爆布局）；控制台底部不再露出系统滚动条，滚轮可横向滚动到最右侧按钮
- 修复命名/删除确认展开时芯片被裁切却无溢出箭头指示
- 修复弹出面板图标在无 emoji 字体环境显示为方块（全部换为矢量图标）
- 修复伴侣编辑窗贴屏幕右缘时可能窄到裁切按钮（最小宽度 320，贴边改为左移）

### 变更

- 归档详情改为轻量渲染（标题/引用/代码块/粗体），不再裸显 Markdown 源码；「复制」仍输出可粘贴的源文
- 归档删除加二次确认（按钮变「确认删除？」，3 秒超时复原）；并发新增快照不再被删除操作覆盖
- 弹出面板「非 AI 站点」提示改中性文案且禁用档位按钮（原为红色警告+可点击）；快捷键列表逐行排版
- 控制台图标按钮按语义分组分隔，「关闭全部」悬停显示危险色；状态点切换加柔和过渡；键盘焦点环随主题
- 失败汇总移至「重试」按钮旁并放宽宽度；归档时间格式跟随界面语言
- 历史归档窗口纳入与伴侣编辑窗同款的受管生命周期：重复点击聚焦既有窗口，随控制台联动最小化/抬前，「关闭全部」一起关闭

## [0.7.1] - 2026-07-11

### 新增

- 新增 ChatGPT GPT-5.6 Sol 适配：切档前先切到目标模型，旧模型（5.3~5.5/o3）高档不再冒充深度思考档
- 新增历史归档：汇总复制/导出时自动定格「问题+各站回答」快照（≤30 条），控制台归档按钮打开只读查看页
- 新增汇总导出：所选站点最新回答保存为 .md 文件（无需新增权限）
- 新增逐站响应耗时标注（状态点悬停可见，如「DeepSeek · 5.9s」），直接服务速度对比
- 新增群发超时自动重试一轮：慢加载站点首开的假性超时自愈，提示如实标注「已自动重试」
- 新增历史提问下拉：鼠标/触屏也能翻历史（与输入框 ↑↓ 并存）
- 新增群发全部成功的读屏播报与芯片状态兜底超时（回调断掉不再永久卡「发送中」）

### 修复

- 修复汇总复制五处保真问题：Markdown 特殊字符转义、标题紧邻代码块被吞（ChatGPT）、Claude/Kimi/千问思考段泄漏进汇总、代码块语言头文本重复、代码内容含反引号时围栏错乱
- 修复 DeepSeek 对话中档位不可读（现优先读 DeepThink 开关）：悬浮控件高亮、巡检、群发前切档确认恢复正常
- 修复通用发送键路径的假成功：点击后同样校验提交（流式中按钮被复用为「停止」等场景诚实报失败可重试）
- 修复「关闭全部」后在途群发的迟到结果把已清空的状态点重新点亮
- 修复重试时窗口已被关闭的站点空等 22 秒（现立即提示「未开窗」）
- 修复多站会话记忆并发覆盖丢失，及发送后 3 秒内点「全部新会话」被复活的竞态
- 修复伴侣编辑窗可被双开
- 修复意外错误在中文界面裸露英文原文（补「未知错误」兜底文案）；「输入框未找到」提示补「未登录」自查方向
- 修复内联命名/删除确认收尾后键盘焦点丢失

### 变更

- 平铺/关闭全部/新会话/重试在后台排队期间显示忙碌禁用态，不再看似无响应
- 「已开窗」改为空心绿圈，与「已回答」的实心绿勾区分（此前平铺后满屏绿勾易误读为已回复）
- 汇总复制的提示如实标注无回答站数；状态点信息同步进无障碍标签
- 弹出面板「打开控制台」在控制台首次使用时预勾当前站点
- 巡检与汇总复制改为并行逐站：耗时从各站之和降为最慢单站
- Claude 档位切换在思考开关缺失时明确报错，不再静默跳过

- 新增回答采集适配（真机锚点审计）：九站全部可取最新回答

### 修复

- 修复新开窗口总是续上旧对话的问题：会话恢复仅限重启后首次开窗

### 变更

## [0.6.2] - 2026-07-06

本版源自一次全仓审查（8 维度深查 + 逐条对抗验证），详见 `docs/audit-2026-07.md`。

### 新增

- 新增一键巡检：只读诊断所选站点适配是否失效，结果标在芯片上
- 新增会话记忆：新开窗口自动续上上次会话，跨重启不丢对比现场
- 新增失败汇总：结果回齐后内联列出失败站点，悬停可看逐站原因

### 变更

- 调整平铺基准为控制台所在显示器，拖到哪屏就铺哪屏
- 调整隐式开窗策略：追问少数站点不再重排或关闭其他窗口
- 调整成功/失败标记为 SVG 勾叉形状，色觉无关且规避字形卡顿
- 调整术语：「广播控制台」统一为「群发控制台」
- 优化失败原因提示：三语文案并附带下一步指引
- 优化 popup 外观，跟随系统暗色模式
- 优化伴侣编辑窗：支持 Ctrl+Enter 发送，空输入抖动提示
- 优化历史浏览：悬停显示「历史 n/N」位置指示
- 优化无障碍：失败结果接入读屏播报，暗色对比度达 WCAG AA
- 优化首次使用提示：芯片与「平铺」按钮补充用途说明

### 修复

- 修复注入失败仍误报发送成功、问题被静默丢失的缺陷
- 修复豆包、Kimi、DeepSeek、千问控件缺失时假报切换成功的缺陷
- 修复档位切换失败仍显示全绿的问题，绿点改附警示说明
- 修复 Gemini 中文界面思考等级被静默跳过的问题
- 修复跨 Chrome 重启窗口登记错乱、可能误关无关弹窗的隐患
- 修复群发进行中关闭控制台后再弹出整批窗口的竞态
- 修复控制台被开进普通标签后抢焦日常窗口的问题
- 修复连按 Alt+Q 双开控制台的问题
- 修复控制台位于副屏时平铺窗口错位的问题
- 修复窄屏下站点芯片被裁切、无法选择的问题
- 修复按 ↑ 浏览历史时覆盖未发送草稿且无法恢复的问题
- 修复伴侣窗发起的群发失败后无法重试的问题
- 修复群发中切换分组后丢失进度状态的问题
- 修复删除分组时当前勾选被覆盖的问题
- 修复控制台与伴侣窗跨窗口编辑时回填失效的问题
- 修复输入法确认候选词误触发命名保存与历史导航的问题
- 修复悬浮胶囊切换显示模式后残留半透明的问题

### 安全

- 固定 release workflow 的第三方 Action 到 commit SHA，防供应链投毒
- 增加打包产物完整性校验，缺文件即打包失败，杜绝坏包发布
