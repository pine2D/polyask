# 更新日志

本项目所有值得注意的变更都记录在此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循[语义化版本](https://semver.org/lang/zh-CN/)并与 `manifest.json` 一致。
发版时 release workflow 提取对应版本段作为 GitHub Release 说明，缺段落拒发。

## [未发布]

## [0.6.3] - 2026-07-07

### 新增

- 新增站点区溢出指示箭头：有隐藏芯片时出现，点击可滚动

### 变更

- 调整站点区滚动方式：隐藏滚动条，支持滚轮横滚与按住拖动

### 修复

- 修复横向滚动条挤占高度、站点芯片被挤偏上的问题
- 修复控制台窗口出现多余垂直滚动条的问题

## [0.6.2] - 2026-07-06

本版源自一次全仓审查（8 维度深查 + 逐条对抗验证），详见 `docs/audit-2026-07.md`。

### 新增

- 新增一键巡检：只读诊断所选站点适配是否失效，结果标在芯片上
- 新增会话记忆：新开窗口自动续上上次会话，跨重启不丢对比现场
- 新增失败汇总：结果回齐后内联列出失败站点，悬停可看逐站原因

### 变更

- 调整平铺基准为控制台所在显示器，拖到哪屏就铺哪屏
- 调整隐式开窗策略：追问少数站点不再重排或关闭其他窗口
- 调整成功/失败标记为 SVG 勾叉形状，色觉无关且规避字形卡顿
- 调整术语：「广播控制台」统一为「群发控制台」
- 优化失败原因提示：三语文案并附带下一步指引
- 优化 popup 外观，跟随系统暗色模式
- 优化伴侣编辑窗：支持 Ctrl+Enter 发送，空输入抖动提示
- 优化历史浏览：悬停显示「历史 n/N」位置指示
- 优化无障碍：失败结果接入读屏播报，暗色对比度达 WCAG AA
- 优化首次使用提示：芯片与「平铺」按钮补充用途说明

### 修复

- 修复注入失败仍误报发送成功、问题被静默丢失的缺陷
- 修复豆包、Kimi、DeepSeek、千问控件缺失时假报切换成功的缺陷
- 修复档位切换失败仍显示全绿的问题，绿点改附警示说明
- 修复 Gemini 中文界面思考等级被静默跳过的问题
- 修复跨 Chrome 重启窗口登记错乱、可能误关无关弹窗的隐患
- 修复群发进行中关闭控制台后再弹出整批窗口的竞态
- 修复控制台被开进普通标签后抢焦日常窗口的问题
- 修复连按 Alt+Q 双开控制台的问题
- 修复控制台位于副屏时平铺窗口错位的问题
- 修复窄屏下站点芯片被裁切、无法选择的问题
- 修复按 ↑ 浏览历史时覆盖未发送草稿且无法恢复的问题
- 修复伴侣窗发起的群发失败后无法重试的问题
- 修复群发中切换分组后丢失进度状态的问题
- 修复删除分组时当前勾选被覆盖的问题
- 修复控制台与伴侣窗跨窗口编辑时回填失效的问题
- 修复输入法确认候选词误触发命名保存与历史导航的问题
- 修复悬浮胶囊切换显示模式后残留半透明的问题

### 安全

- 固定 release workflow 的第三方 Action 到 commit SHA，防供应链投毒
- 增加打包产物完整性校验，缺文件即打包失败，杜绝坏包发布

### 变更

- 升级 Claude 快速档到 Sonnet 5（思考开 + Effort Medium）

## [0.5.0] - 2026-06-29

### 新增

- 新增界面三语支持（English、简体中文、繁體中文）

### 变更

- 调整打开控制台快捷键为 Alt+Q（原 Alt+A）

### 修复

- 修复窗口前后台联动在 Windows 下时灵时不灵的问题
- 修复智谱清言思考档位适配（站点改版为弹层菜单）
- 修复群发结果误标红与一批注入判定细节

## [0.4.0] - 2026-06-27

### 新增

- 新增分组下拉，合并预设与自定义分组
- 新增命名模板与发送历史（↑/↓ 翻阅）
- 新增伴侣编辑窗，长文本与细条双向同步
- 新增发送进度聚合与失败站一键重试
- 新增 Alt+A 快捷键打开群发控制台
- 新增发送后自动置顶全部窗口（popup 可关闭）
